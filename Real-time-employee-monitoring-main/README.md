## 💡 What I built:
A Real-Time Webcam-Based Productivity Monitoring System that can:
- Detect if the user is present in front of the screen
- Monitor eye closure (for drowsiness or sleep detection)
- Track mouth movement (for excessive talking detection)
- Automatically log working status to a CSV file (with time & duration)

## 🛠️ Technologies & Techniques Used:
- MediaPipe Face Mesh for 3D facial landmark tracking
- Eye Aspect Ratio for blink detection
- Mouth Aspect Ratio for talking detection

## 📌 Why this matters:
This system is ideal for:
- Remote employee monitoring
- Study & focus tracking
- for catching the occasional “kaam chor” employee (don’t take too seriously!)
